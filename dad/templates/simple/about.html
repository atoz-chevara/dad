About this site

The main purpose of this website is to allow people to thank package maintainers and developers for making Debian possible. It was inspired by the "Debian Appreciation Day" created in 2010 by Margarita Manterola, Maximiliano Curia, Valessio Brito and Raphael Geissert.

This version was written by Lincoln de Sousa e Marcelo Jorge Vieira and has some novelties such as users' location and images sharing. Besides bringing knowledge about how Debian users are spread around the world it will also feed a repository of free images that can be used for promoting Debian in the future.

The source code is available at http://git.comum.org/dad.git.
